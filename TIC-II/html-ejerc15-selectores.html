Tipos de selectores que entran en examen

Clase (class="")

Los selectores de clase son herramientas que, como su nombre lo indica, permiten seleccionar todos los elementos que tienen un mismo nombre de clase.

Identificador (id="")

Un selector de ID está diseñado para seleccionar elementos con base en su atributo de ID.

Todos los elementos (*)

El selector universal (*) selecciona todos los elementos HTML en la página.

Varios elementos ( p, em, h1, h2)

El selector de agrupación selecciona todos los elementos HTML con el mismo estilo.

Elementos contenidos en otros elementos (ol li)

Las propiedades CSS de las listas son las que nos permiten controlar los estilos de los marcadores y la posición de los elementos dentro de las listas.

Sobrevuelo (p:hover)

El selector :hover se utiliza para seleccionar elementos cuando pasa el mouse sobre ellos.
